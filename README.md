🥇 React-Game-Project
Rock Scissors Paper

React 기초 문법을 익히고, 컴포넌트와 prop,state를 활용한 가위바위보 게임입니다.
컴퓨터와 가위바위보 게임을 진행하여, 승무패를 가릴 수 있으며, 해당 게임의 스코어점수를 지정할 수 있습니다.
